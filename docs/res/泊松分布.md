泊松分布（Poisson distribution）用来描述：

**在固定时间、固定面积或固定空间里，某类事件发生了多少次。**

前提是这类事件有几个特点：

1. 事件是**随机**发生的
2. 平均发生率基本**稳定**
3. 两次事件彼此**独立**
4. 在非常短的时间内，通常只会发生 **0 次或 1 次**，很少同时发生多次

---

## 一、直观理解

比如：

* 一小时内客服接到多少个电话
* 一分钟内网站收到多少次访问
* 一页书里出现多少个错别字
* 一段公路上每天发生多少起事故

这些都不是“最多几次里成功几次”的问题，而是“某段范围里自然发生多少次”的问题，这就很适合用泊松分布。

---

## 二、定义

如果随机变量 (X) 表示某个区间内事件发生的次数，且
(X \sim \text{Poisson}(\lambda))，那么它的概率公式是：

[
P(X=k)=\frac{\lambda^k e^{-\lambda}}{k!}, \quad k=0,1,2,\dots
]

其中：

* (k)：实际发生的次数
* (\lambda)：该区间内的**平均发生次数**
* (e)：自然常数，约等于 2.71828

---

## 三、参数 (\lambda) 是什么意思

(\lambda) 很重要，它既表示：

* **均值**
* 也表示**方差**

也就是说：

[
E(X)=\lambda,\quad Var(X)=\lambda
]

例如，若某路口平均每小时发生 3 次车辆违章，那么：

* 均值 = 3
* 方差 = 3

---

## 四、举个例子

某呼叫中心平均每分钟接到 2 个电话，设 (X) 为某一分钟接到的电话数。
那么：

[
X \sim \text{Poisson}(2)
]

### 1）这一分钟恰好接到 0 个电话的概率

[
P(X=0)=\frac{2^0 e^{-2}}{0!}=e^{-2}\approx 0.1353
]

### 2）恰好接到 3 个电话的概率

[
P(X=3)=\frac{2^3 e^{-2}}{3!}=\frac{8e^{-2}}{6}\approx 0.1804
]

### 3）最多接到 2 个电话的概率

[
P(X\le 2)=P(0)+P(1)+P(2)
]

分别算：

[
P(0)=e^{-2}\approx 0.1353
]

[
P(1)=2e^{-2}\approx 0.2707
]

[
P(2)=\frac{2^2 e^{-2}}{2}=2e^{-2}\approx 0.2707
]

所以：

[
P(X\le 2)\approx 0.1353+0.2707+0.2707=0.6767
]

---

## 五、为什么它常用

因为很多现实问题只知道“平均每单位发生多少次”，这时泊松分布很方便。

例如知道：

* 平均每天 5 个故障
* 平均每小时 12 位顾客进店
* 平均每平方米 3 个瑕疵点

就可以直接建模。

---

## 六、和二项分布的关系

泊松分布可以看作**二项分布的极限情况**。

如果：

* 试验次数 (n) 很大
* 单次发生概率 (p) 很小
* 但 (np=\lambda) 保持有限

那么二项分布

[
Bin(n,p)
]

可以近似成泊松分布

[
Poisson(\lambda)
]

这叫做**泊松近似**。

### 例子

一批产品有 1000 个，每个次品率 0.002。
次品个数原本可看作二项分布：

[
X\sim Bin(1000,0.002)
]

因为：

* (n=1000) 很大
* (p=0.002) 很小
* (np=2)

所以可近似：

[
X\sim Poisson(2)
]

---

## 七、泊松分布的图像特点

* ( \lambda ) 小时，分布偏右，0、1、2 次最常见
* ( \lambda ) 变大时，图形逐渐更对称
* 当 ( \lambda ) 很大时，泊松分布还可以进一步近似正态分布

---

## 八、什么时候不适合用

泊松分布不适合下面这些情况：

* 事件之间**不独立**
* 发生率会随时间明显变化
* 一次会“成批”出现很多事件
* 总次数有明确上限，更像“成功/失败重复试验”

例如：

* 节假日商场客流量随时间变化很大
* 疫情传播中一个人可能带来连锁感染，不独立
* 考试题做对几道，更适合二项分布

---

## 九、一个简单记忆法

你可以这样区分：

* **二项分布**：做了 (n) 次，每次成不成功
* **泊松分布**：一段时间/区域里，自然发生了几次

一句话记：

**“二项分布看试验次数，泊松分布看事件次数。”**

---

## 十、总结

泊松分布适合描述“单位区间内随机事件发生次数”。

核心公式：

[
P(X=k)=\frac{\lambda^k e^{-\lambda}}{k!}
]

核心特征：

* 只有一个参数 (\lambda)
* 均值和方差都等于 (\lambda)
* 常用于稀有事件、到达次数、缺陷数等问题
